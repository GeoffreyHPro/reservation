# ⚡️ Standard Execution

Start postgresql database

```bash
docker-compose up
```

Destroy database

```bash
docker-compose down -v
```

Start API REST

```bash
mvn spring-boot:run
```

URL of Swagger: http://localhost:8080/swagger-ui.html
